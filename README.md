## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/AdinotAadi/Aditya-Patel---Portfolio.git

2. Install dependencies:
   ```bash
   cd portfolio-react-tailwind
   npm install
3. Run the development server:
   ```bash
   npm start
